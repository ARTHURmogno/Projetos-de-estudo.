# 📚 Projetos de Estudos

Este repositório foi criado com o objetivo de armazenar meus projetos de estudo e acompanhar minha evolução como desenvolvedor Java.

## 🎯 Objetivos

- Praticar Programação Orientada a Objetos (POO)
- Aprender e aplicar boas práticas de desenvolvimento
- Desenvolver projetos utilizando Java
- Praticar acesso a banco de dados com JDBC
- Aprender SQL e modelagem de dados
- Estudar estruturas de dados e collections
- Aprender Spring Boot e APIs REST
- Construir um portfólio de projetos

## 🚀 Tecnologias

- Java
- JDBC
- MySQL
- Git
- GitHub
- Spring Boot (em estudo)

## 📈 Evolução

Este repositório será atualizado conforme novos projetos forem desenvolvidos e novos conhecimentos forem adquiridos.
